## Instructions d'installation
Ce projet ne nécessite aucune installation complexe car il utilise Tailwind CSS via un lien direct (CDN).

1. Clonez ce dépôt sur votre machine :
   `git clone [https://github.com/killerange7-lang/Projet-Tailwind-2026.git]`
2. Accédez au dossier du projet.
3. Ouvrez le fichier **index.html** dans votre navigateur internet pour visualiser le site.
